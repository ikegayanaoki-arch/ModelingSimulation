Modeling & Simulation Lecture Material

Environmental Settings

Please build python environment in you local computer to fully understand the contents of this lecture. Though I again explain how to install and run simulation codes, I recommend you to build it before the lecture. 

For macOS users:

0. Go to "https://brew.sh"

1. Copy URL from home-brew website 

2. Type “brew search python”

3. Type “brew install python??”

*Change ?? to the version number you find by the "brew search" command.

4. Launch python by 

  -> python3 filename.py



For Windows users:

0. Go to https://www.anaconda.com/products/individual

1. Install Anaconda.

2. Open Anaconda Navigator

3. Launch "Jupyter Notebook"

4. Move to the directory you saved a source code.

5. Click "New" and select Python 3 / Click "New", select "Notebook", and select Python3 (Try either of them; it depends on the version).

6. Write "exec(open("filename.py").read())" in the first line

7. Click "Run>"

Tips

<For windows users who cannot use Jupiter (I was faced with a case where Jupiter did not work)>

1) Please install python3 for windows from https://www.python.org/downloads/windows/

2) Start "power shell" from your startup menu.

3) Move to the directory you have filename.py

4) Run following command.

-> py -3 filename.py

*To install package, use pip by

-> py -3 -m pip install "package name (such as numpy, matplotlib etc.)"



<For students who got error by ImageMagic unavailable message>

Windows: This seems to occur for PC which doesn't have "ImageMagick" package (This is irrelevant to Anaconda installation). So, please go to ImageMagick site, and download/install the package. This probably occurred for other students, so let me share it with all other students, not but with students who asked me questions.  Site: https://www.imagemagick.org/script/download.php, and download ImageMagick (download from the link, or search ImageMagick by google, and download the latest version for windows).


MacOS: Install "Imagmagick" by brew. Type "-> brew install imagemagick"



<Jupyter notebook>

To use animation, add "%matplotlib notebook" at the first line.

環境設定

この講義の内容を十分に理解するために、各自のローカルPCでPython環境を構築しておいてください。シミュレーションコードのインストール方法や実行方法は講義中にも再度説明しますが、できれば講義前に準備しておくことを勧めます。

macOS を使用する場合:

0. 「https://brew.sh」を開く

1. Homebrew のWebサイトからURLをコピーする

2. 「brew search python」と入力する

3. 「brew install python??」と入力する

*「??」の部分は、「brew search」コマンドで見つかったバージョン番号に置き換えてください。

4. 次のようにして Python を起動する

  -> python3 filename.py



Windows を使用する場合:

0. https://www.anaconda.com/products/individual を開く

1. Anaconda をインストールする

2. Anaconda Navigator を開く

3. 「Jupyter Notebook」を起動する

4. ソースコードを保存したディレクトリへ移動する

5. 「New」をクリックして Python 3 を選ぶ / または「New」から「Notebook」を選び、Python3 を選ぶ

6. 1行目に「exec(open("filename.py").read())」と書く

7. 「Run>」をクリックする

補足

<Jupyter を使えない Windows ユーザー向け（実際に Jupyter が動作しないケースがありました）>

1) https://www.python.org/downloads/windows/ から Windows 用 Python3 をインストールしてください。

2) スタートメニューから「PowerShell」を起動してください。

3) filename.py があるディレクトリへ移動してください。

4) 次のコマンドを実行してください。

-> py -3 filename.py

*パッケージをインストールする場合は、pip を使って次のように実行してください。

-> py -3 -m pip install "package name (such as numpy, matplotlib etc.)"



<ImageMagic unavailable のエラーが出た学生向け>

Windows: このエラーは、「ImageMagick」パッケージが入っていないPCで起こるようです（Anaconda のインストール有無とは無関係です）。ImageMagick のサイトからダウンロードしてインストールしてください。同じ問題が起こる学生がほかにもいる可能性があるため、質問した学生だけでなく全体にも共有しておきます。サイト: https://www.imagemagick.org/script/download.php


MacOS: brew で「Imagemagick」をインストールしてください。

-> brew install imagemagick



<Jupyter Notebook>

アニメーションを使うには、1行目に「%matplotlib notebook」を追加してください。
