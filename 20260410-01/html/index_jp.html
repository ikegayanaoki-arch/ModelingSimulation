4. モデリングとシミュレーションの数学的基礎
- ベクトルと座標
- 3 次元直交座標 (x, y, z) と速度成分 (u, v, w) を使用。太字はベクトルを表す。内積と外積は既知として進める。
- 微分演算子と表記
- 支配方程式は偏微分方程式として与えられる。ナブラ ∇ の各成分を ∂i と書き、ラプラシアン ∇² は ∂i∂i と表せる。
- 勾配はスカラー φ に対する (∂i φ)、発散は ∂i ui でスカラー、回転は εijk ∂j uk でベクトルになる。
- スカラー量とベクトル量
- 内積、ラプラシアン、発散はスカラー。
- 外積、勾配、回転はベクトル。
- 行列とテンソル
- 行列成分は Aij、単位行列は E。
- ベクトルの外積 xi yj は 9 成分の 2 階テンソルになり、内積 x・y と区別する必要がある。
- 速度勾配テンソル ∇u は 2 階テンソル、発散 ∂i ui はスカラーである。
- テンソル表記と添字
- ベクトルを xi（i=1,2,3）、行列を aij と書くことで導出や記述が簡潔になる。
- クロネッカーのデルタとレヴィ・チヴィタ記号
- δij は i=j のとき 1、それ以外は 0。
- εijk は偶置換で 1、奇置換で -1、添字が重なれば 0。
- アインシュタインの総和規約
- 同じ添字が 1 項に 2 回現れたら総和を意味する。例: xi yi = x1y1 + x2y2 + x3y3。
- 2 回現れる添字はダミー添字、1 回だけのものは自由添字。
- 添字の縮約と恒等式
- δ を使うと xi δij → xj のように添字を縮約できる。
- δ と ε の関係を使うと外積や回転の式変形がしやすくなる。
5. 添字表記で書く連続の式とナビエ・ストークス方程式
- 連続の式
- 発散項は ∂j uj と簡潔に書ける。蓄積項も添字表記で書ける。
- 非圧縮（密度一定）の場合は ∂j uj = 0。ダミー添字は i, j, k のどれでもよい。
- ナビエ・ストークス方程式の各項
- 3 つの運動量方程式を添字表記で 1 つの形にまとめられる。
- 時間微分は ∂t ui、移流項はダミー添字を使って簡潔に表され、圧力勾配は ∂i p、拡散項は ∂j ∂j ui と表される。
- 展開形と比較することで、各項の物理的意味を理解できる。

要約

この講義では、熱と流体運動の物理的基礎として、輸送現象における基本的な支配方程式を扱った。まず、離散的な分子を連続体として扱う連続体仮定を説明し、それによって 3 次元空間の任意の点で速度、温度、濃度といった巨視的物理量を定義できることを示した。次に、蓄積項・移流項・拡散項からなる一般輸送方程式を提示し、質量保存（連続の式）、運動量保存（ニュートン流体に対するナビエ・ストークス方程式）、内部エネルギー保存（熱方程式）の収支式を導出した。目的は、今後の講義でこれらの方程式を解くための理論的基礎を与えることである。最後に、Python 環境を整え、配布されたスクリプトを実行して動作確認するという実践課題が示された。

知識ポイント

1. 輸送現象の基本概念
- 授業後半の主題
- 後半では、熱、内部エネルギー、流体運動の支配方程式の物理を扱う。
- 次週以降は、これらの方程式を実際に解いていく。
- 三次元場の解析
- 室内の温度分布のような三次元場の現象を対象にする。
- 3 次元直交座標系 (x, y, z) で場の任意点を表す。
- その点で速度 (u, v, w)、温度 T、濃度 C などの物理量を定義できる。
- これらは巨視的状態量として扱う。
- 連続体仮定
- 支配方程式導出の出発点。
- 実際には分子は離散的でランダムに運動しているが、1 モルあたり約 10^23 個という膨大な粒子数のため、連続体として近似できる。
- その結果、ある点の速度や温度を、多数の分子の平均的ふるまいを表す単一の値として扱える。
2. 一般輸送方程式
- 一般形
- 任意の輸送量 φ に対する支配方程式は次のように書ける。
- ∂φ/∂t + uⱼ(∂φ/∂xⱼ) = D(∂²φ/∂xⱼ²) + F
- ここで t は時間、xᵢ は空間座標、D は拡散係数 (m²/s)、F はソース項またはシンク項。
- 各項の意味
- 第1項: 蓄積項
- ∂φ/∂t は固定点での時間変化を表す。
- 第2項: 移流項
- uⱼ(∂φ/∂xⱼ) は流体のバルク運動による輸送を表す。
- 第3項: 拡散項
- D(∂²φ/∂xⱼ²) は分子のランダム運動による高いところから低いところへの輸送を表す。
- 普遍性と適用先
- この形は温度、速度、濃度などに共通して現れる。
- 温度では D は熱拡散率 α。
- 運動量では D は動粘性係数 ν で、さらに圧力項が加わる。
- 授業では、まず単純な拡散方程式から始め、後で移流を含む問題へ進む。
3. 収支式の導出
- 方法: 検査体積法
- 小さな固定領域（コントロールボリューム）を取り、その中への流入、流出、蓄積を考えて支配方程式を導出する。
- 連続の式（質量保存）
- 質量保存を表す。
- 流入・流出・蓄積をつり合わせることで導かれる。
- 運動量収支式
- ニュートンの第2法則「運動量変化率 = 力」に基づく。
- 蓄積項: 検査体積内の運動量（質量×速度）の時間変化。
- 移流項: 流れによって運ばれる運動量で、ρu², ρuv, ρuw などの項が現れる。
- 作用する力
- 圧力: 各面の圧力差による力。
- 粘性力（応力）: 内部摩擦による力。応力 τ は単位面積あたりの力であり、τyx は y 面に作用する x 方向の力を表す。
- これらをまとめた一般形が運動量の収支式である。
- ナビエ・ストークス方程式
- 一般の運動量方程式には未知の応力テンソル τij が含まれる。
- ニュートン流体では、応力は速度勾配に比例するという粘性法則を用いる。
- これを代入すると、CFD ソフトウェアが解くナビエ・ストークス方程式が得られる。
- 内部エネルギー収支式（熱方程式）
- 熱力学第一法則「内部エネルギー変化 = 正味の熱 + 仕事」に基づく。
- 蓄積項: 検査体積内の内部エネルギーの時間変化。
- 熱伝導（拡散）: フーリエの法則に従い、温度勾配に比例する熱流束で表される。
- 移流項: 流体運動によって運ばれるエネルギー。
- 圧力仕事や粘性散逸を含む一般形から、よく使う温度輸送方程式へ簡略化される。
4. 重要な支配方程式の整理
- 流体と熱に関する主要 3 方程式
1. 連続の式: 質量保存
2. ナビエ・ストークス方程式: 運動量保存
3. 温度（内部エネルギー）方程式: エネルギー保存
- 拡散係数の考え方
- 拡散項はいずれも類似した数学的形を持つ。
- その係数が diffusivity（拡散率）である。
- 運動量では動粘性係数 ν。
- 熱では熱拡散率 α。
- いずれも単位は m²/s で、どれだけ広がりやすいかを表す。
