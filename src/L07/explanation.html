This lesson covers a program that simulates cavity flow by solving the incompressible Navier-Stokes equations using the SMAC (Simplified Marker and Cell) method.

Execution Results (from demo)

The simulation is run for different Reynolds numbers (Re). As Re increases, the flow becomes more complex, and secondary vortices appear in the corners.

Code Explanation (cavityAns.py)

The core of the SMAC method involves a three-step process within each time loop: Prediction, solving the Poisson equation for pressure, and Correction.

    # Calculate an intermediate velocity (u*, v*) without considering the pressure gradient term.
    # This velocity field does not satisfy the continuity equation.

            v[j,i] = (...) # Navier-Stokes without pressure term
    
    # --- 2. Pressure Poisson Equation ---
    # Iteratively solve the Poisson equation for the pressure correction 'p'.
    # The source term of this equation is the divergence of the intermediate velocity field.
    pp[:,:]=0.0
    flg = 1
    while flg == 1:
       flg=0
       for i in range(1,N+1): 
           for j in range(1,N+1): 
               # Solve for pressure 'p'
               p[j,i] = ( ... - (( u[j,i+1] - u[j,i] )*dl + ( v[j+1,i] - v[j,i] )*dl )/dt ) / cd[j,i]
       
       # ... (Convergence Check) ...
       pp=cp.copy(p)
    
    # --- 3. Correction Step ---
    # Correct the intermediate velocity using the calculated pressure gradient
    # to obtain the final velocity for the time step, which satisfies the continuity equation.
    for i in range(2,N+1):
        u[1:N,i] = u[1:N,i] - dt*(p[1:N,i]-p[1:N,i-1])/dl 
        v[i,1:N] = v[i,1:N] - dt*(p[i,1:N]-p[i-1,1:N])/dl

    # Update pressure and store old velocities for the next time step
    p[:,:]=pn[:,:]+p[:,:]
    uo=cp.copy(u)
    vo=cp.copy(v)

コード解説 (cavityAns.py)

SMAC法の核心は、各時間ステップにおける「予測子」「圧力のポアソン方程式」「修正子」の3つのステップです。

# ... (パラメータと格子の設定は省略) ...

# 時間発展ループ
for n in range(NO):
    # --- 1. 予測子 (Prediction) ステップ ---
    # 圧力勾配項を無視して、中間的な速度 (u*, v*) を計算する。
    # この速度場は、連続の式を満たさない。
    for i in range(1,N+1):
        for j in range(1,N+1):
            u[j,i] = (...) # 圧力項を無視したナビエ・ストークス方程式
            v[j,i] = (...) # 圧力項を無視したナビエ・ストークス方程式
    
    # --- 2. 圧力のポアソン方程式 ---
    # 圧力補正量 'p' に関するポアソン方程式を反復計算で解く。
    # この方程式のソース項は、中間速度の発散 (div u*) である。
    pp[:,:]=0.0
    flg = 1
    while flg == 1:
       flg=0
       for i in range(1,N+1): 
           for j in range(1,N+1): 
               # 圧力 'p' を計算
               p[j,i] = ( ... - (( u[j,i+1] - u[j,i] )*dl + ( v[j+1,i] - v[j,i] )*dl )/dt ) / cd[j,i]
       
       # ... (収束判定) ...
       pp=cp.copy(p)
    
    # --- 3. 修正子 (Correction) ステップ ---
    # 計算された圧力勾配を使って中間速度を修正し、
    # 連続の式を満たす最終的な速度場を得る。
    for i in range(2,N+1):
        u[1:N,i] = u[1:N,i] - dt*(p[1:N,i]-p[1:N,i-1])/dl 
        v[i,1:N] = v[i,1:N] - dt*(p[i,1:N]-p[i-1,1:N])/dl

    # 次の時間ステップのため、圧力の更新と古い速度の保存を行う
    p[:,:]=pn[:,:]+p[:,:]
    uo=cp.copy(u)
    vo=cp.copy(v)
