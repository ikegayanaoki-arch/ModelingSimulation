  1. 支配方程式の復習（熱と流体）

熱伝導方程式（一般形と簡略形）

非定常形: rho cp ∂T/∂t = lambda ∇²T（1次元では rho cp ∂T/∂t = lambda ∂²T/∂x²）。

定常形（時間変化なし）: ∇²T = 0（1次元では ∂²T/∂x² = 0）。

記号: 温度は大文字 T、時間は t、lambda は熱伝導率、rho cp は密度と比熱。

2次元熱伝導

非定常: rho cp ∂T/∂t = lambda (∂²T/∂x² + ∂²T/∂y²)。

定常: ∂²T/∂x² + ∂²T/∂y² = 0。定常でも T = T(x,y) なので偏微分を用いる。

連続の式と Navier–Stokes 方程式（2次元）

連続の式: ∂u/∂x + ∂v/∂y = 0。

x方向運動量式（u式）: ∂u/∂t + u ∂u/∂x + v ∂u/∂y = −(1/rho) ∂p/∂x + nu (∂²u/∂x² + ∂²u/∂y²)。

y方向運動量式（v式）: ∂v/∂t + u ∂v/∂x + v ∂v/∂y = −(1/rho) ∂p/∂y + nu (∂²v/∂x² + ∂²v/∂y²)。

nu は動粘性係数を表し、圧力勾配は各方向に対応する。

テンソル（添字）表記による一般形

連続の式: ∂Ui/∂Xi = 0（i = 1..3、繰り返し添字は総和）。

Navier–Stokes: ∂Ui/∂t + Uj ∂Ui/∂Xj = −(1/rho) ∂P/∂Xi + nu ∂²Ui/∂Xj∂Xj。

内部エネルギー式（温度の移流拡散形）: ∂T/∂t + Uj ∂T/∂Xj = alpha ∂²T/∂Xj∂Xj。alpha は熱拡散率で、圧力項を除けば運動量式と同じ構造を持つ。

概念的な構造: 移流拡散

蓄積項（非定常項）、移流項、拡散項は、温度や濃度のようなスカラー量でも、速度のようなベクトル量でも共通する基本構造を持つ。圧力項は運動量式にのみ現れる。

教育上の流れとしては、まず純粋な拡散（定常）から始め、次に時間依存項、さらに移流項を加え、最後に完全な Navier–Stokes へ進む。

小規模な棒の実験

ステンレス、銅、アルミニウムの棒にパラフィンを載せ、熱がどのように伝わるかを可視化し、サーモグラフィで熱の浸透を確認した。

観察結果: 銅は熱の伝達が速く、ステンレスは遅い。表面のパラフィンはおよそ40〜50℃程度で溶け、棒に沿った温度分布を示す。

大規模な棒の実験

条件: 直径約10cm、長さ約20mの銅棒を用い、一端を強力なバーナーで加熱。

時間発展: 約18〜20時間加熱しても、熱源近傍では銅が溶けそうになるほど高温になる一方、表面温度の上昇は数メートル（約3m）程度しか進まず、20m先までは到達しない。

温度分布: 表面温度と位置の関係として示され、室温は約20℃。分布は線形ではなく下に凸の形となり、周囲への熱損失の重要性を示している。測定されているのは表面温度であり、内部温度は異なる可能性がある。

モデル化上の解釈

現実的な再現には、周囲空気への熱損失（対流）と境界条件の考慮が必要である。表面損失を考えない純粋熱伝導だけでは、実験で見た下に凸の温度分布と一致しない線形分布しか得られない。

  1. 今日扱う一次元定常熱伝導の問題設定

熱損失項を含む支配方程式

出発点となる PDE: rho cp ∂T/∂t = lambda ∂²T/∂x² − Q。ここで Q は熱損失を表す（Q > 0 なら損失）。

定常化: 非定常項を落として整理すると ∂²T/∂x² = Q/lambda となる。

境界条件（両端温度固定）

x=0 で T = TL（低温側、例 25℃）。

x=L で T = TH（加熱側、例 1000℃）。

例として長さ L = 20m を想定。

比較する3つの熱損失ケース

ケース1: 断熱（完全断熱）

物理仮定: 表面から熱が出ていかない（たとえば発泡スチロールのような断熱材で覆う）。このとき Q = 0。

支配 ODE: ∂²T/∂x² = 0 → 温度分布 T(x)=Ax+B の線形分布。

含意: この線形分布は実験で観測された下に凸の分布と一致しない。

ケース2: 位置に依存しない一定の対流的熱損失

単純化した仮定: 棒全体の代表的温度差（たとえば「最高温度−最低温度」）に比例する一定の熱損失として扱い、一定係数 H を用いた対流を抽象化する。Q は x に依存しない定数とする。

支配 ODE: ∂²T/∂x² = Q/lambda（Q は一定）→ 二次関数的な温度分布。

狙い: 位置依存を入れずに非線形な分布形状を表現すること。

ケース3: 位置依存の対流熱損失（より現実的）

物理モデル: 局所的な対流熱損失は、局所表面温度と周囲温度との差に比例する。q_conv(x)=h [T(x)−TL]。h は熱伝達係数（単位 J·m⁻²·s⁻¹·K⁻¹）で、流体種別・流れ・形状に依存する伝熱効率を表す。

支配 ODE: ∂²T/∂x² = (h/lambda)[T(x)−TL]。右辺に T が含まれる線形常微分方程式であり、高温側ほど損失が大きくなる下に凸の分布を表現できる。

注: h は空気か水か、自然対流か強制対流かなどで変わり得るが、ここでは扱いやすさのため一定とする。

  1. FDMとFVMの基本概念と相違

FDM

微分を格子上の差分で近似（テイラー展開に基づく中央差分など）。一様格子で簡明。複雑形状では座標変換が必要。

FVM

商用ソフトで一般的。制御体積で積分し、ガウスの発散定理で体積積分を境界積分へ還元。非一様・複雑格子に柔軟。

同一メッシュでの整合

一様格子・同サイズではFDMとFVMの最終形が一致する場合があるが、一般には導出と扱いが異なる。

  1. ガウスの発散定理の適用と非一様格子の勾配評価（FVM）

二階微分の低減

制御体積積分により、Tの二階微分は東西境界の一次導関数（勾配）差に置換。

東側Eの勾配

(Ti+1−Ti) / [0.5·(Δx(i+1)+Δx(i))]。一様格子で(…)/Δx。

西側Wの勾配

(Ti−Ti−1) / [0.5·(Δx(i)+Δx(i−1))]。一様格子で(…)/Δx。

源項の体積積分

q/lambda一定仮定で q/lambda · Δx(i) に変換。

  1. FVM離散方程式の係数化と最終式

係数定義

CE=1/[0.5·(Δx(i+1)+Δx(i))]、CW=1/[0.5·(Δx(i)+Δx(i−1))]、CO=CE+CW。

最終更新式

Ti = (1/CO)·[ CE·Ti+1 + CW·Ti−1 − (q/lambda)·Δx(i) ]。

連立構造と境界

i=1..Nに対し同形のN式。T0, Tn+1は既知境界。暗示的形のため反復解法を用いる。

  1. 反復法（ヤコビ法）の原理とTiへの適用

固定点反復

g(x)=0 を x=f(x) に変形し、xk+1=f(xk) で更新、収束判定で停止する。

Ti更新の実装

予測値Pを用い、Ti = (1/CO)[ CE·T(i+1)^P + CW·T(i−1)^P − (q/lambda)·Δx(i) ] と更新する。

i=1..N を更新後、T と予測値の差で収束判定を行い、未収束なら予測値を更新して繰り返す。

準備
